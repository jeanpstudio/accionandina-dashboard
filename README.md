# 🏔️ Andean Monitoring Hub | Enterprise Operations & Audit Dashboard

A high-performance, bespoke internal infrastructure engineered to centralize regional operations, partner auditing, and high-altitude storytelling for a multi-national conservation program across 5 South American countries.

## 🎯 The Mission
Acción Andina faced a "visibility gap" between massive ecological impact in the field and centralized operational oversight. I architected this ecosystem to bridge the gap between technical reforestation data and emotional narratives, providing real-time supervision of 15+ international partners.

## 🚀 Key Modules & Engineering

- **🔍 Custom Audit & Compliance Engine**: A robust module designed to track tasks, campaign progress, and partner performance in real-time. Includes an automated "Red-Flag" system to identify operational delays before they impact donor reporting.
- **☁️ Real-Time Data Infrastructure**: Powered by **Supabase**, ensuring secure authentication, instant data synchronization, and a relational PostgreSQL backbone for complex operational auditing.
- **📦 Global Story Bank (Media Lab)**: A centralized repository for high-end cinematography and field narratives. It standardizes global communications, ensuring consistent high-quality output regardless of the partner's location.
- **✉️ Strategic Mailing Lab**: A bespoke internal tool for donor communication, allowing the organization to professionalize outreach with data-backed narratives and automated workflows.

## 🛠 Technical Stack

- **Frontend**: [React 18+](https://react.dev/) (Dynamic, state-driven UI for real-time monitoring).
- **Build Tool**: [Vite](https://vitejs.dev/) (Optimized for rapid development and high-performance production builds).
- **Backend & Database**: [Supabase](https://supabase.com/) (Auth, PostgreSQL, and Real-time subscriptions).
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for a clean, data-heavy administrative interface.
- **Architecture**: Modular "Command Center" design, optimized for multi-tenant scalability across 15+ organizations.

## 🏗 System Architecture Highlights

- **Full-Stack Integration**: Leveraged Supabase's real-time capabilities to provide instant updates on partner task completion and content uploads.
- **Bespoke ERP Logic**: Unlike off-the-shelf tools, this system was built from the ground up to match the specific auditing needs of reforestation grants.
- **Data Transparency**: 100% visibility for international donors (e.g., GFG, UN) into the monthly content production and web maintenance of regional partners.
